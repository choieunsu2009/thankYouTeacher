# 스승의 날 기념 롤링페이퍼 사이트

## 시스템 명세

- FE : html, css, js로 아주 간단히 구현할것
- BE : node.js(express js 기반)으로 구현할것
- DB : mysql을 사용하고 docker로 관리할것
- 배포 : github actions를 사용하여 자동으로 배포할것
- repo : 단일 레포로 구성하여 복잡도를 최소화 할것

## 디자인 명세

- (학생 화면.png), (교사 화면.png)를 와이어프레임 삼아 작업하고 명세된 모든 기능을 만족하는 최고의 UI/UX를 구성할것
- 카네이션 아이콘은 (카네이션.png)를 이용할것
- 깔끔하고 귀엽게 디자인하되, 불편할 정도로 요소가 다양하거나 복잡한 UI/UX를 피할것
- 교사 화면에서 이미지를 로딩할 때 이미지의 크기에 맞게 적당히 변동되어야 하나, 최대, 최소 크기를 규정하고 최소/최대를 넘어선 크기의 이미지면 fill할것
- 폰트는 pretendard와 paperlogy를 이용할것

## 기능명세

- 학생을 총 15명이고 각각 랜덤한 고유 식별 번호를 부여하여 관리할것
- 한 학생당 최대 3개의 글을 작성 할 수 있다
- 한 학생당 작성한 글은 교사 화면에서 확인할 수 있다
- 학생은 이미지 입력이 선택적으로 가능하며, 이미지를 입력하지 않을 시 (카네이션.png)를 기본 이미지로 사용할것
- 전달할 말의 최대 길이는 400자이다
- 전달하기 버튼을 누르면 먼저 개인식별번호가 올바른지 확인 후 이후 작업을 수행할것
- 만약 개인식별번호가 올바르지 않으면 alert를 띄우고 이후 작업은 수행하지 말것

- 교사 화면 접근 라우트는 '/teacherSYJ'로 접근 가능하다
- 학생 화면 접근 라우트는 '/'로 접근 가능하다
- 교사 화면에서는 전체 화면을 png로 저장 할 수 있다 (버튼을 만들것)
- 교사 화면에서는 각 학생이 전송한 글과 이미지를 각각 개별로 png 저장이 가능하다

- 이 사이트의 도메인은 'thankyou.a-end.kr' 이며 CNAME으로 등록되어있다.
- 배포할 상황을 대비하여 localhost로 하드코딩은 금지된다

- 백엔드는 restAPI를 이용하여 데이터를 주고받을것
